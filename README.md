# webflow-code-editor README

This extension allows Webflow developers to edit their custom code blocks on their sites using Visual Studio Code.

This is not an official Webflow product. 

Created by [Raymond Camden](https://www.raymondcamden.com).

## Features

* Allows you to edit the code used in the head and footer of your site.
* For each page under your site, you can also edit code in the head and footer. 

## Requirements

Must have a Webflow account and published site.

## Known Issues

At this time you can't edit code blocks *inside* pages. That will come later.
